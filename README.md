# srl-sandbox

Sandboxed development environments on macOS using [Apple Container](https://github.com/apple/container). Spin up isolated Linux containers for safe coding with LLM agents like [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

Each container runs in its own lightweight VM with sub-second startup, direct IP networking, and proper filesystem isolation.

## Install

### Via Homebrew (recommended)

```bash
brew tap DCPMA/srl-sandbox https://github.com/DCPMA/srl-sandbox.git
brew install srl-sandbox
```

### Manual

```bash
git clone https://github.com/DCPMA/srl-sandbox.git
cd srl-sandbox
./install.sh
```

## Quick Start

```bash
# Build the container image (first time only, ~60s)
srl-sandbox build

# Launch a sandbox for the current project
cd ~/Projects/myapp
srl-sandbox
```

The CLI stays attached in the foreground. Press **Ctrl+C** to stop the container. Use `-d` to detach.
